# Funnel Templates

Repository di progetto per **Funnel Templates**, parte dell'ecosistema Bentivoglio Consulting.

## Cosa contiene questo progetto

Il progetto Funnel Templates ha tre direttrici strategiche:

1. **Strategia Principale**, costruzione di una directory di 50 template HTML pronti all'uso, integrati con Systeme.io
2. **Strategia 1 (Outbound Proattivo, in corso)**, identificare siti da rifare, ricostruirli con AI, inviarli in outbound per portare clienti su Systeme.io
3. **Strategia 2 (Fase 2)**, app custom che permette al cliente di inserire il suo URL e ricevere il sito rifatto automaticamente
